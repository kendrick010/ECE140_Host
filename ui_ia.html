Product UI & IA



Pep Pets integrates three tacticle buttons for navigating through the OLED screen menu. These buttons have a familiar appearance to buttons on hand-held game devices, indicating a "move left", "move right", and "select/enter" functionality.

Pep Pets's IA is inspired by Dan Brown's Principle of Choices and Disclosure, as the main the screen only displays the common and utmost options, the feed and show menu. If the user desires to select other Pep Pet features, such as accessing the friend's list, foodbar, etc, the user can always select the show menu button. Thus, the Pep Pet's interface will ressemble an hierarchial structure as selection screens will be overlaid from the most recent/important selection from the user.
